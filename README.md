# Local Prospect Audit Landing Page

This is the V1 validation landing page for **Local Prospect Audit**, built using Next.js (App Router), Vanilla CSS, and Vercel.

## Project Structure
* `app/page.js`: Main landing page containing all 10 wireframe sections.
* `app/globals.css`: Modern, minimal design system and custom styling.
* `app/layout.js`: Global HTML shell with custom SEO metadata.

## Setup & Running Locally
1. Install dependencies (already completed):
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is intended to be deployed on Vercel.

Steps:
1. Push this project to GitHub.
2. Import the repository into Vercel.
3. Deploy with default Next.js settings.
4. Add a custom domain later if needed.
